## Memulai

### Registrasi Akun

**Langkah-langkah:**

## Top-Up Akun Iklan

Fitur untuk menambah saldo ke akun iklan Anda (Facebook, Google, TikTok) melalui transfer bank atau crypto.

### Cara Top-Up:

1. **Pilih Menu "Top-Up"** dari sidebar
2. **Pilih Akun Iklan** yang ingin di top-up dari dropdown
3. **Masukkan Jumlah Top-Up**
   - Pilih mata uang (IDR atau USD)
   - Ketik nominal (contoh: 5000000 untuk Rp 5 juta)
4. **Pilih Metode Pembayaran**
   - **Bank Transfer**: Transfer ke rekening bank
   - **Crypto**: Transfer menggunakan cryptocurrency
5. **Upload Bukti Pembayaran**
   - Klik tombol "Upload Bukti Transfer"
   - Pilih file (JPG, PNG, atau PDF)
   - File maksimal 5MB
6. **Klik "Submit Top-Up Request"**

### Setelah Submit:

1. Request Anda akan masuk status **"Pending"**
2. Admin akan melakukan verifikasi
3. Anda akan menerima **notifikasi email dan in-app** saat:
   - ‚úÖ Top-up disetujui (saldo ditambahkan ke akun)
   - ‚ùå Top-up ditolak (dengan alasan)

### Upload Proof (Bukti Spend Limit & Budget Aspire)

Setelah top-up disetujui admin, Anda perlu upload 2 bukti dari dashboard akun iklan Anda:

**Untuk Facebook Ads:**
1. **Spend Limit Proof** (Bukti Batas Pengeluaran)
   - Screenshot halaman spend limit dari Facebook Ads Manager
2. **Budget Aspire Proof** (Bukti Budget Aspire)
   - Screenshot halaman budget aspire dari Facebook Ads Manager

**Untuk Google Ads & TikTok Ads:**
1. **Spend Limit Proof** saja (Budget Aspire tidak diperlukan)

**Cara Upload:**
1. Buka menu **"Riwayat Pembayaran"**
2. Cari top-up request yang sudah disetujui
3. Klik tombol **"Upload Proofs"**
4. Upload file sesuai kebutuhan
5. Klik **"Upload"**

**Penting:** Upload proof ini wajib dilakukan untuk menyelesaikan proses top-up.

## Top-Up Wallet

Fitur untuk mengisi saldo wallet Anda melalui transfer bank atau crypto. Saldo wallet bisa digunakan untuk transfer ke akun iklan.

### Cara Top-Up Wallet:

1. **Pilih Menu "Top-Up Wallet"** dari sidebar
2. **Pilih Jenis Wallet**
   - **Main Wallet**: Wallet utama untuk operasional
   - **Withdrawal Wallet**: Wallet khusus hasil penarikan
3. **Pilih Mata Uang**
   - IDR (Rupiah)
   - USD (Dollar)
4. **Masukkan Jumlah**
   - Ketik nominal yang ingin di top-up
5. **Pilih Metode Pembayaran**
   - Bank Transfer
   - Crypto
6. **Upload Bukti Transfer**
   - Klik tombol upload
   - Pilih file bukti pembayaran
7. **Klik "Submit Request"**

### Status Request:

- **Pending**: Menunggu verifikasi admin
- **Approved**: Disetujui, saldo ditambahkan ke wallet
- **Rejected**: Ditolak oleh admin

Anda akan menerima notifikasi email dan in-app untuk setiap perubahan status.

## Transfer Wallet ke Akun

Fitur untuk mentransfer saldo dari wallet Anda ke akun iklan tanpa perlu upload bukti pembayaran.

### Cara Transfer:

1. **Pilih Menu "Transfer Wallet"** dari sidebar
2. **Pilih Sumber Wallet**
   - Main Wallet IDR
   - Main Wallet USD
   - Withdrawal Wallet IDR
   - Withdrawal Wallet USD
3. **Pilih Akun Tujuan**
   - Pilih akun iklan yang ingin menerima transfer
4. **Masukkan Jumlah**
   - Ketik nominal yang ingin ditransfer
   - Pastikan saldo wallet mencukupi
5. **Klik "Submit Transfer"**

### Catatan Penting:

- ‚úÖ Transfer instan jika disetujui admin
- ‚úÖ Tidak perlu upload bukti pembayaran
- ‚úÖ Mata uang wallet dan akun harus sama
- ‚ùå Transfer tidak bisa dibatalkan setelah disetujui

## Permintaan Akun

Fitur untuk mengajukan permintaan akun iklan baru (Facebook, Google, atau TikTok).

### Cara Mengajukan Permintaan:

1. **Pilih Menu "Permintaan Akun"** dari sidebar
2. **Pilih Platform**
   - Facebook Ads
   - Google Ads
   - TikTok Ads
3. **Isi Informasi Akun**
   - **Nama Akun**: Nama untuk identifikasi akun
   - **Account ID**: ID akun dari platform (contoh: Facebook Business Manager ID)
   - **Mata Uang**: IDR atau USD
4. **Isi Informasi Tambahan** (sesuai platform)
   - Untuk Facebook: Business Manager ID, Ad Account ID
   - Untuk Google: Customer ID
   - Untuk TikTok: Business Center ID
5. **Klik "Submit Request"**

### Proses Setelah Submit:

1. Status awal: **"Pending"**
2. Admin akan review request Anda
3. Status berubah menjadi:
   - **"Approved"**: Disetujui, masuk proses share
   - **"In Progress"**: Sedang diproses share oleh admin
   - **"Completed"**: Share selesai, akun siap digunakan
   - **"Rejected"**: Ditolak (dengan alasan)

Anda akan menerima notifikasi email untuk setiap perubahan status.

## Penarikan Saldo

Fitur untuk menarik saldo dari akun iklan Anda ke wallet penarikan.

### Cara Melakukan Penarikan:

1. **Pilih Menu "Penarikan"** dari sidebar
2. **Pilih Akun Iklan** yang ingin ditarik saldonya
3. **Masukkan Jumlah**
   - Ketik nominal yang ingin ditarik
   - Atau pilih **"Tarik Semua"** untuk withdraw full balance
4. **Klik "Submit Withdrawal"**

### Proses Penarikan:

1. Status awal: **"Pending"**
2. Admin akan verifikasi saldo aktual di akun
3. Admin upload 2 bukti:
   - Bukti saldo sebelum penarikan
   - Bukti saldo setelah penarikan
4. Admin input jumlah terverifikasi
5. Status berubah:
   - **"Approved"**: Saldo dipindahkan ke withdrawal wallet
   - **"Completed"**: Penarikan selesai
   - **"Rejected"**: Ditolak (dengan alasan)

### Catatan:

- ‚úÖ Saldo hasil withdraw masuk ke **Withdrawal Wallet**
- ‚úÖ Mata uang mengikuti akun (IDR/USD)
- ‚úÖ Jumlah terverifikasi bisa berbeda dengan request (sesuai saldo aktual)

## Riwayat Pembayaran

Halaman khusus untuk melihat riwayat top-up akun iklan yang Anda lakukan.

### Informasi yang Ditampilkan:

**Setiap Top-Up Request:**
- Reference Code (Kode unik)
- Platform & Nama Akun
- Jumlah
- Mata Uang
- Metode Pembayaran (Bank/Crypto)
- Status
- Tanggal Request
- Tanggal Verifikasi (jika sudah diverifikasi)

### Status Top-Up:

1. **Pending** (Menunggu)
   - Top-up belum diverifikasi admin
   - Bukti pembayaran sudah diupload

2. **Verified** (Terverifikasi)
   - Admin sudah verifikasi
   - Saldo sudah ditambahkan ke akun
   - **Action Required**: Upload Spend Limit & Budget Aspire proof

3. **Completed** (Selesai)
   - Semua proof sudah diupload
   - Top-up selesai sepenuhnya

4. **Rejected** (Ditolak)
   - Top-up ditolak oleh admin
   - Lihat alasan penolakan di detail

### Aksi yang Tersedia:

**Untuk Status "Verified":**
- **Upload Proofs**: Upload bukti spend limit dan budget aspire

**Untuk Semua Status:**
- **Lihat Detail**: Melihat detail lengkap top-up
- **Download Bukti**: Download bukti pembayaran yang sudah diupload

## FAQ

### Pertanyaan Umum

**Q: Berapa lama proses verifikasi top-up?**
A: Biasanya 1-24 jam kerja. Anda akan menerima notifikasi email dan in-app saat top-up diverifikasi.

**Q: Apakah saya bisa top-up di luar jam kerja?**
A: Ya, Anda bisa submit top-up kapan saja. Admin akan memproses saat jam kerja.

**Q: Bagaimana cara melihat saldo akun iklan saya?**
A: Buka menu "Kelola Akun" untuk melihat saldo semua akun Anda.

**Q: Apakah bisa top-up menggunakan wallet?**
A: Ya, gunakan fitur "Transfer Wallet ke Akun" untuk top-up dari wallet tanpa upload bukti pembayaran.

**Q: Berapa minimal top-up?**
A: Tidak ada minimal top-up. Anda bisa top-up mulai dari nominal kecil sesuai kebutuhan.

**Q: Apakah bisa withdraw sebagian saldo saja?**
A: Ya, Anda bisa input nominal yang ingin ditarik, atau pilih "Tarik Semua" untuk full withdrawal.

**Q: Kemana saldo hasil withdraw masuk?**
A: Saldo withdraw masuk ke **Withdrawal Wallet** Anda sesuai mata uang akun.

**Q: Apakah bisa transfer antar wallet?**
A: Saat ini tidak ada fitur transfer antar wallet. Wallet transfer hanya dari wallet ke akun iklan.

**Q: Format file apa yang diterima untuk upload bukti?**
A: Format yang diterima: JPG, PNG, PDF. Maksimal ukuran file 5MB.

**Q: Perbedaan Main Wallet dan Withdrawal Wallet?**
A: 
- **Main Wallet**: Untuk top-up dan operasional
- **Withdrawal Wallet**: Khusus menerima hasil penarikan dari akun iklan

**Q: Apakah ada biaya admin?**
A: Hubungi admin untuk informasi biaya dan kebijakan.

**Q: Bagaimana cara menghubungi support?**
A: Gunakan fitur notifikasi atau hubungi admin melalui dashboard.
